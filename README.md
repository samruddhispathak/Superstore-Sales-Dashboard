# Superstore-Sales-Dashboard- Excel<br>
Interactive Excel Sales Dashboard analysing Superstore retail data with KPIs, PivotTables and dynamic slicers.
# 📊 Project Overview
An interactive, multi-sheet Excel Sales Dashboard built on the sample Superstore retail dataset. Designed to demonstrate advanced Excel skills, including data modelling, PivotTables, dynamic charts, and business insight generation.

## 🎯 Objectives
- Analyse £2.3M+ in sales data across regions, categories and segments
- Identify profit drivers and loss-making areas
- Build an interactive dashboard with dynamic slicers
- Derive actionable business insights from raw data

  ## 🛠️ Tools & Techniques Used
- **Microsoft Excel** (Advanced)
- PivotTables & PivotCharts
- Dynamic Named Tables (`SalesData`)
- SUMIF, AVERAGEIF, COUNTIF, UNIQUE, SUMPRODUCT
- Combo Charts (Column + Line with dual axis)
- Interactive Slicers connected to all PivotTables
- KPI Cards with linked formulas
- Custom number formatting & conditional formatting

## 📁 File Structure
| Sheet | Purpose |
|---|---|
| `RAW_DATA` | Original protected source data |
| `CALCULATIONS` | All KPI formulas & analysis |
| `PIVOT_DATA` | 6 PivotTables & 6 PivotCharts |
| `DASHBOARD` | Final interactive dashboard |

# 📈 Key Business Insights
- **Technology** is the most profitable category (£145,455 profit)
- **Furniture** has high sales but the lowest profit margin — Tables & Bookcases are loss-making
- **West region** leads in both sales (£725,458) and profit (£108,418)
- **Standard Class** shipping accounts for 59% of all orders
- Discounts above 20% consistently result in **negative profit margins**
- **California** is the top state with £457,688 in sales

  ## 🖼️ Dashboard Preview
The dashboard includes:
- 4 KPI cards (Total Sales, Total Profit, Profit Margin, Avg Discount)
- Interactive slicers for Region, Category and Segment
- 6 dynamic charts updating in real time
